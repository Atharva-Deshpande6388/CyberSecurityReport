#Project: SQL Injection (SQLi) Vulnerability Analysis

##File: SQL-Injection-Report.pdf

###Target: testphp.vulnweb.com (A designated, public website for security testing)

Project Overview

This report documents a successful manual SQL injection attack performed on the login page of the designated test website, testphp.vulnweb.com. The objective of this exercise was to identify and exploit an SQL injection vulnerability to bypass the authentication mechanism.


Note: All activities were conducted in a controlled, educational environment for skill-building and demonstration purposes.
